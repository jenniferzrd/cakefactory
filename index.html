Programmation Orientée Objet (POO)

En JS, les constructeurs sont de simples fonctions, exécutées avec le mot-clé "new".
On fabrique un moule à objects (le constructeur).
Dns le scope de ce moule, on indique les variables propres à chaque object construit.
On y place également la logique de fonctionnement initiale (comment notre objet va se comporter).

Avec le prototype, on associe des fonctions propres à chaque objet construit.
On appelle ces fonctions des "méthodes d'instance".
Le but du jeu est de modulariser notre code, de fabriquer des outils autonomes.
On les combine pour construire nos applications.
